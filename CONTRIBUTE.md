# Manual de contribución

Si quieres contribuir a este proyecto, puedes seguir las siguientes instrucciones:

## Si apenas iniciaste

Ejecuta estos pasos si quieres hacer tu primera contribución, estos pasos solamente se harán la primera vez:

1. Lee el archivo `README.md` para entender el propósito del proyecto.
2. Clona el repositorio a tu máquina local usando:
   ```
   git clone https://github.com/TechAtlasDev/tamagotchi.git
   ```
3. Entra al directorio del proyecto:
   ```
   cd tamagotchi
   ```
4. Haz las modificaciones que consideres necesarias en el código.
5. Añade los archivos modificados al área de preparación:
   ```
   git add .
   ```
6. Realiza un commit con un mensaje descriptivo:
   ```
   git commit -m "Descripción de los cambios realizados"
   ```
7. Sube tus cambios al repositorio remoto:
   ```
   git push origin main
   ```

## Si ya has contribuido antes

Sigue estos pasos si ya has hecho contribuciones anteriormente:

1. Asegúrate de estar en el directorio del proyecto:
   ```
   cd tamagotchi
   ```
2. Actualiza tu repositorio local con los últimos cambios del repositorio remoto:
   ```
   git pull origin main
   ```
3. Haz las modificaciones que consideres necesarias en el código.
4. Añade los archivos modificados al área de preparación:
   ```
   git add .
   ```
5. Realiza un commit con un mensaje descriptivo:
   ```
    git commit -m "Descripción de los cambios realizados"
   ```
6. Sube tus cambios al repositorio remoto:
   ```
   git push origin main
   ```
